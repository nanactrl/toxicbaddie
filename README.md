# Cyberbullying Detection Discord Bot

## Overview

This project presents a Cyberbullying Detection Discord Bot developed as a Final Year Project (FYP). The system utilizes Machine Learning and Natural Language Processing (NLP) techniques to identify and classify cyberbullying messages in Discord servers.

The bot monitors user messages in real time and predicts whether a message contains harmful, toxic, insulting, threatening, or cyberbullying content. By integrating a trained machine learning model with the Discord platform, the system helps create a safer online environment and supports automated content moderation.

## Features

- Real-time message monitoring on Discord
- Cyberbullying and toxicity detection
- Machine Learning-based text classification
- NLP preprocessing pipeline
- Automated moderation and warning system
- Performance evaluation using multiple ML algorithms

## Technologies Used

- Python
- Discord.py
- Scikit-learn
- Pandas
- NumPy
- Natural Language Processing (NLP)
- Machine Learning
- Git & GitHub

## Project Objective

To develop an intelligent Discord bot capable of detecting cyberbullying behaviour through machine learning and NLP techniques, helping online communities maintain a safer and more respectful communication environment.
